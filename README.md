# 🍔 今天吃什么？

一个帮助大学生决定吃什么的 3D 交互网页，用神经网络般的三维可视化展示食物，附带热量劝退功能。

## ✨ 功能

- 🔮 **3D 玻璃球体** — Three.js 构建，可拖拽旋转、缩放
- 🌡️ **热量颜色渐变** — 黄→橙→红，低卡到高卡一目了然
- 🧩 **食材自动拆解** — 100+ 种食物支持食材逐层分解
- 🎡 **转盘随机** — 带运动模糊效果的旋转转盘
- 🗄️ **350+ 种食物数据库** — 覆盖 14 大菜系
- 🌐 **在线搜索** — 接入 Open Food Facts 全球数据库
- ➕ **自定义添加** — 支持手动添加食物并自动归类
- 📱 **响应式设计** — 手机和电脑都能用

## 🛠️ 技术栈

- Three.js 0.157 — 3D 渲染
- Open Food Facts API — 在线食物搜索
- 纯 HTML/CSS/JS — 单文件，零依赖构建

直接打开 `index.html` 即可，无需安装任何东西。
